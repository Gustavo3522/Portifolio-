Projeto em Java

Sistema de Gerenciamento de Funcionários Este projeto consiste em uma aplicação console em Java que gerencia informações de funcionários.

Funcionalidades principais:
- Cadastro de funcionários: Registra nome, função, salário e data de admissão.
- Exibição de dados: Visualiza detalhes de todos os funcionários cadastrados.
- Atualização: Permite alterar informações de um funcionário específico.
- Exclusão: Remove um funcionário da lista de registros.

A aplicação utiliza conceitos de orientação a objetos e listas dinâmicas, sendo ideal para aprendizado e prática de manipulação de dados em Java. Projeto desenvolvido para aprimorar conhecimentos e desenvolver novas habilidades Ver Projetos

Projeto em HTML/CSS

Agência de Viagens - Site Promocional
Este projeto é um site promocional responsivo para uma agência de viagens, desenvolvido com HTML, CSS e JavaScript.

Funcionalidades principais:
- Apresentação de pacotes de viagem: Listagem de destinos com informações detalhadas e promoções.
- Serviços oferecidos: Tours guiados, fotografia profissional e aluguel de veículos.
- Seção de contato: Informações da agência com integração ao Google Maps.
- Interatividade: Navegação suave por âncoras e ícones otimizados com Feather Icons.

O site é ideal para agências de turismo que desejam atrair clientes apresentando seus pacotes e serviços de forma moderna e atrativa. Projeto Proposto pela Universidade Católica de Brasília Ver Projetos

Projeto em C++

Este projeto em linguagem C consiste em um sistema robusto e completo de bilheteria de cinema, projetado para gerenciar de forma eficiente todo o processo de vendas de ingressos. Desde a reserva de lugares nas salas de cinema até o controle financeiro, o sistema foi desenvolvido com o objetivo de simular um ambiente real utilizado em cinemas. Ele busca não apenas automatizar tarefas manuais, mas também fornecer uma experiência intuitiva e organizada para os usuários.

Funcionalidades principais:
- Escolha de filmes e assentos disponíveis.
- Compra de ingressos com opção de meia-entrada e combos.
- Cancelamento de reservas com ajuste no total arrecadado.
- Geração de relatórios financeiros.
- O projeto explora conceitos de estrutura de dados, manipulação de matrizes
- e interação com o usuário por meio de menus simples e eficientes.

Projeto Proposto pela Universidade Católica de Brasília Ver Projetos

Projetos em Phyton

Loja Virtual - Simulador de Pagamento
Este projeto é uma aplicação em Python que simula um sistema de pagamento para uma loja virtual. Ele permite ao usuário calcular o preço final de uma compra com base na forma de pagamento escolhida.

Funcionalidades principais:
- Cálculo de descontos: Aplicação de descontos para pagamentos à vista em dinheiro, cheque ou cartão.
- Parcelamento sem juros: Opção de parcelamento em 2x no cartão, sem acréscimos.
- Parcelamento com juros: Parcelamento em 3x ou mais no cartão, com acréscimo de 20% sobre o valor total.
- Validação de entrada: Tratamento de opções inválidas de pagamento.

O projeto reforça a prática de estruturas condicionais e cálculos financeiros básicos, sendo ideal para aprendizado em lógica de programação e interação com o usuário. Ver Projetos .
